# Architecture Decision Records (ADR)

## Purpose

Architecture Decision Records document the important technical decisions made during the development of LAIR.

Rather than relying on memory or commit history, each significant decision is captured in its own document.

This provides:

- Historical context
- Design rationale
- Alternatives considered
- Consequences
- Future references

---

## ADR Format

Each ADR contains:

- Status
- Context
- Decision
- Alternatives
- Consequences
